# 🎮 Tic-Tac-Toe Pro: Web & Python Edition

A dual-platform Tic-Tac-Toe game featuring a modern Web interface and a robust Python command-line version. Both versions utilize the **Minimax Algorithm** to provide a challenging "Hard" mode that is mathematically impossible to beat.

## Features
* **Dual Platforms:** Play in your browser or your terminal.
* **Smart AI:** Choose between Easy, Medium, and Hard (Minimax) difficulties.
* **Modern Design:** Web version features a "Glassmorphism" UI with smooth animations.
* **Local PvP:** Play against a friend locally on either version.

## Project Structure
* `index.html` - The structural skeleton of the web game.
* `style.css` - Modern styling and layout.
* `script.js` - Game logic and AI for the browser.
* `tictactoe_pro.py` - Object-Oriented Python version of the game.

## How to Run

### 1. Web Version
No installation required! 
1. Open the project folder.
2. Double-click `index.html` to open it in your favorite web browser.
3. Use this link  "https://yuvi1245.github.io/Tic-Tac-Toe-with-AI/"
   

### 2. Python Version
Requires Python 3.x installed.
